3093 → V2 Converter

How it works

Upload one or more 3093 SLA detail CSV exports. The converter reads the German column headers, applies the Compare_Dir breach rule (dir=1: breach when numerator < denominator; dir=−1: breach when numerator > 0), and outputs a V2-structured .xlsx file with one sheet per SLA code — identical to the format expected by Breaches Tracker and Combined Analyzer.

✅ Incident Ticket, Queue, ISO_Language, TOPIC mapped directly ✅ DATE_CLOSE, Priority, Status, Tool mapped directly ✅ One sheet per SLA code (KSL-2a, KSL-3a, KM-1, KSL-4 …) ✅ Manual columns (Agent, BMS ID, Comment …) left blank — ready to fill ✅ All tickets from the official export accepted — no ticket format filtering ✅ Rows outside Mar–May or missing SLA code go to a _Excluded sheet — no data is lost
